# 🌀 Speedback (Frontend)

Welcome to the frontend for Speedback!

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Vitest + React Testing Library

## 🚀 Getting Started

**1. Install the magic**

```bash
npm install
```

**2. Start the fun**

```bash
npm run dev
```

**3. Run the tests (because I care!)**

```bash
npm run test
```
